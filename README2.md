# 🛠 Мои блоки для FLProg

Коллекция пользовательских блоков для среды **FLProg**, созданная для упрощения разработки проектов на Arduino.

## 📝 Описание
В этом репозитории я храню свои наработки, которые редактирую в **VSCode**. Блоки оптимизированы для стабильной работы и снабжены комментариями внутри кода.

### Основные возможности:
* **Поддержка контроллеров:** Arduino Uno, Nano, ESP8266 (укажи свои).
* **Чистый код:** Минимум лишних библиотек.
* **Гибкая настройка:** Все важные параметры вынесены в пользовательское меню блока.

---

## 🚀 Как использовать
1. Скачайте нужный файл блока (обычно это `.ubi` или архив).
2. В программе **FLProg** перейдите в меню `Блоки пользователя` -> `Библиотека блоков`.
3. Нажмите `Импортировать блок` и выберите скачанный файл.

---

## 📂 Структура репозитория
* `/Blocks` — готовые файлы блоков для импорта.
* `/Source` — исходный код C++ (для ознакомления и доработки в VSCode).
* `/Examples` — примеры проектов `.flp`.
